exit

protected void exit(boolean saved)
Called when the user cancels or has successfully saved. This default implementation tells the PlaceController to show the details of the edited record, or clears the display if a creation was canceled.

If we're creating, a call to getRecord() from here will return a record with the correct id. However, other properties may be stale or unset.

Parameters:
saved - true if changes were comitted
